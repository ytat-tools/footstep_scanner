# footstep_scanner
【直下掘りチェッカー】

Minecraft用の鉱石探索ツールです。

## 使用方法
アイテム「直下掘りチェッカー」を手に持った状態で
・スニーク＋長押し：ラフスキャン
・通常長押し：詳細スキャン

## 取得方法
/give @s ytat:footstep_scanner
あるいはクリエで道具の区分から…
※レシピは後日実装予定！
